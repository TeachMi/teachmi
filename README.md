# TeachMe App

Product application scaffold for TeachMe. Planning and UX source documents remain in `../design`.

## Requirements

- Node.js `>=20.9.0`
- pnpm `10.33.3`

## Commands

Run from this directory:

```bash
pnpm dev
pnpm build
pnpm lint
pnpm start
```

The UI is Hebrew-first and RTL from the root layout. Later BASE stories add the full app shell, design tokens, providers, auth, Storybook, and E2E harness.
